# Users - Angular web application
## Summary ☀️
 **MEAN stack web app**
a web application that displays users data and allow CRUD
this is **FullStack** project that uses the technologies:
* Node.js - Backend
* EXPRESS - Server
* Angular - Frontend (Client)
* MongoDB - Database
